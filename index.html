Introdução

Python é uma das linguagens de programação mais populares e de crescimento mais rápido do mundo. Ele é usado para todos os tipos de tarefas, incluindo programação web e análise de dados, e surgiu como a linguagem a ser aprendida para aprendizado de máquina. Essa popularidade significa que os desenvolvedores Python estão em demanda, e os trabalhos de programação Python podem ser lucrativos.

As razões acima mencionadas são por que pode ser uma boa ideia para você aprender a programar em Python. Este módulo fornecerá uma introdução ao uso do Python para construir um aplicativo, que pode ser um ponto de partida para se tornar um programador Python.

O que é o Python?

O Python é uma das linguagens de programação mais populares do mundo. Criado no início da década de 1990, o Python pode ser empregado para uma ampla gama de usos, desde automatizar tarefas repetitivas e escrever aplicativos web até construir modelos de aprendizado de máquina e implementar redes neurais. Pesquisadores, matemáticos e cientistas de dados, em particular, gostam do Python por causa de sua sintaxe rica e fácil de entender e da ampla gama de pacotes de código aberto disponíveis. Os pacotes são bibliotecas de código compartilhado que estão disponíveis gratuitamente para qualquer pessoa usar.

Python tem uma sintaxe simples e fácil de aprender, que enfatiza a legibilidade. As aplicações escritas em Python podem ser executadas em quase todos os computadores, incluindo os que utilizam Windows, macOS e distribuições populares do Linux. Além disso, o ecossistema contém um conjunto avançado de ferramentas de programação que permitem escrever, depurar e publicar aplicações Python.

Finalmente, o Python é suportado por uma comunidade de usuários ativa que está ansiosa para ajudar novos programadores a aprender a maneira Pythonic, onde você não apenas obtém a sintaxe certa, mas usa a linguagem da maneira pretendida.

Executar código em Python

Python é uma linguagem interpretada , o que reduz o ciclo edit-test-debug porque não há nenhuma etapa de compilação necessária. Para executar aplicações Python, precisa de um ambiente/interpretador de runtime para executar o código.

A maioria dos ambientes de runtime suporta duas formas de executar código em Python:

  • Modo Interativo: Neste modo, cada comando digitado é interpretado e executado imediatamente, e você vê os resultados cada vez que pressiona Enter. O modo interativo é o modo padrão se você não passar um nome de arquivo para o intérprete.
  • Modo de script: o modo de script, você coloca um conjunto de instruções Python em um arquivo de texto com uma extensão .py. Em seguida, pode executar o interpretador python e direcioná-lo para o ficheiro. O programa é executado linha por linha, e a saída é exibida.
